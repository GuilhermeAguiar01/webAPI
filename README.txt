LEIA!

Requisitos:

1- Ter instalado o JDK com suas respectivas variaveis de ambiente

2- Ter o Eclipse, IntelliJ, Netbeans ou outra IDE de sua preferencia

3- Ter o Maven instalado com suas respectivas variaveis de ambiente

4 - Instalar Cucumber

5 - Copiar o driver com versao compativel com o seu navegador para a pasta driver

Links para download:

1 - JDK - https://www.oracle.com/java/technologies/javase/javase-jdk8-downloads.html

2 - IDE
2.1 - Eclipse - https://www.eclipse.org/downloads/
2.2 - Intellij - https://www.jetbrains.com/pt-br/idea/download/#section=windows
2.3 - Netbeans - https://netbeans.org/features/java/index_pt_BR.html


3 - Maven - https://maven.apache.org/download.cgi 


Exemplo de variaveis de ambiente:

Local de instalacao Java: C:\Program Files\Java\jdk1.8.0_251

Local de instalacao Maven: C:\maven

Variaveis de ambiente -
JAVA_HOME - C:\Program Files\Java\jdk1.8.0_251
MAVEN_HOME - C:\maven
Path - %JAVA_HOME%\bin;%MAVEN_HOME%\bin
